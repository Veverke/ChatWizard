# ChatWizard

A VS Code extension that reads AI chat session data (GitHub Copilot Chat, Claude Code) directly from the local file system and builds a unified, searchable index over accumulated conversation history.

## Features

- Unified search across all AI chat sessions
- Session management panel
- Code block extraction
- Prompt library
- Export to Markdown
- Analytics & usage stats

## Requirements

VS Code 1.85.0 or higher.
